What is this?

This website hosts an image classifier. It can tell you if an uploaded image is either lasagna or a doom level. To try it, simply navigate to the website's home page, upload an image of lasagna or a doom screenshot and click submit. The website should return (hopefully) correct results! If you don't have images of lasagna or doom screenshots on hand, an image search on your favorite search engine will be your best friend.

How does it work?

THE WEBSITE FRONTEND OF HTML5 IS HOSTED VIA PYTHON FLASK LIVING WITHIN A DOCKER CONTAINER HOSTED ON EC2 AWS. THE BACKEND BINARY CLASSIFIER IS A TENSORFLOW INFERENCE MODEL RUNNING IN SERVER MODE, HOSTED THE SAME WAY.

Phew... that was a lot of buzzwords. But seriously, whether you're a budding web developer, or a curious data scientist, if you'd like to know more, the whole stack's source is available as a repository on my Github. You can find a link to that on the left side of the page footer, at the bottom of any page on this website, or directly in the middle of the contact page.
